# React Native with Typescript

RN에서 typescript를 이용하여 프로젝트를 개발해 본다.

## Getting Start

타입스크립트를 많이 사용해봤다는 가정하에 기본적인 문법은 생략하고, 적용 위주로 봐본다.

```bash
$ npx react-native ReactNativeWithTypescript init --template react-native-template-typescript
```
